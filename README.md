# clippy
Makes copy paste easy!

## Installation

1. Clone the repo.

   `git clone https://github.com/offensivelearning/clippy.git`
   
2. Change into the `clippy` folder.
   ` cd clippy`
   
3. Install the requirements.
   `pip install -r requirements.txt`
   
4. Run the program
   ` python clippy.py`
